# gifypet-dell

A fork of [gifypet](https://gifypet.neocities.org/) with some additional features. Mostly worked on for fun for by boyfriend [Dell's](https://dancingdawgz.neocities.org/) neocities.

## Features
- Customization no longer required as GET parameters
- Added playerName
- Added a settings file for customizing various aspects
- Persistent mood, belly, and coins
  * Mood and belly decay while away (at a lowered rate)
- Responses and various statements are saved in a unified string file, so you can adjust statements
  * Auto-randomizing statements
  * Variables can be called in the string
- *Party harder!!*
